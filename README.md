## Regras gerais

- Realizar um projeto por dia
- Compartilhar o progresso nas mídias sociais (Twitter, Facebook, Linkedin etc...), utilizando a hashtag #30DiasDeCSS
- O desafio diário deverá ser concluído até 23:59
